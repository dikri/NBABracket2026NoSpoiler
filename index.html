What is this?

An interactive app to track the 2026 NBA Playoffs at your own pace. You enter results yourself — no spoilers, no stress.

How it works

  • + Record a win · − Undo a win
  • Play-In: first to 1 · Playoffs: first to 4 (Best of 7)
  • Winners advance automatically through all rounds

📅 Watch Date & Countdown

Set the date you've watched through. A countdown shows how many days until you're caught up to live.

🎮 Game Tracker

GAME 1-7 boxes turn green as you enter wins.

🔥 Player Heads

  • Star headshots appear once both teams are set
  • 🔥 Fire glow = series winner · Red = eliminated · 💤 = resting

🔊 Sounds

+1 → squeaks, swish, music riffs · -1 → brick, groan · Series win → fanfare · Championship → full package

Export / Import

📤 Export copies bracket + watch date. Send to other device, 📥 Import there. 🔄 Reset clears all.

Install as app

📱 iPhone: Safari → Share → "Add to Home Screen" · 💻 Chrome: Install icon · 📱 Landscape mode recommended!
